# Currency Converter

A simple Java project that converts one currency to another using exchange rates.

Features

- Convert INR to USD
- Convert USD to INR
- Convert INR to EUR
- Convert EUR to INR
- Easy menu-based program
- Takes input from the user through the console

Technologies Used

- Java
- Scanner Class
- Switch Case

How to Run

1. Download or clone the project.
2. Open it in your Java IDE or VS Code.
3. Compile the program.
4. Run the "CurrencyConverter.java" file.
5. Enter your choice and amount to convert.
