# This is a simple ATM Interface project made in Java. The project allows users to perform basic banking operations such as checking balance, depositing money, and withdrawing money.

The project is developed using Object-Oriented Programming (OOP) concepts like classes, objects, methods, and encapsulation.

Features

- Check account balance
- Deposit money
- Withdraw money
- Validate user input
- Show appropriate messages for transactions

Technologies Used

- Java
- OOP Concepts
- Scanner Class

Project Structure

BankAccount Class

Used to store account balance and perform deposit and withdrawal operations.

ATMmachine Class

Handles user interaction and displays the ATM menu.

ATMInterface Class

Contains the main method to run the program.

How to Run

1. Open the project in VS Code, Eclipse, or IntelliJ IDEA.
2. Compile the Java file.
3. Run the program.
4. Select options from the ATM menu.

What I Learned

- Creating classes and objects
- Using methods
- Encapsulation
- Taking user input
- Validating data
- Building a menu-driven application

Conclusion

This project helped me understand the basics of Java OOP and how real-world systems like an ATM can be implemented using classes and objects.
