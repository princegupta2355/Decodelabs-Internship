# Java Programming Internship Projects

These projects were completed during my internship at DecodeLabs. Each project helped me practice Java programming concepts such as loops, conditions, methods, user input, and object-oriented programming.

Projects

1. Number Game

A simple number guessing game where the user tries to guess a randomly generated number.

Features

- Random number generation
- User input handling
- Hint messages
- Score tracking

2. Student Grade Calculator

A program that calculates the total marks, percentage, and grade of a student.

Features

- Takes marks as input
- Calculates percentage
- Displays grade based on performance
- Simple and easy-to-use interface

3. ATM Interface

A console-based ATM application that allows users to perform basic banking operations.

Features

- Check balance
- Deposit money
- Withdraw money
- Menu-driven system

4. Currency Converter

A Java application that converts one currency into another.

Features

- Multiple currency conversions
- User-friendly menu
- Fast and simple calculations

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Loops and Conditional Statements
- Switch Case

Learning Outcomes

Through these projects, I improved my understanding of:

- Java fundamentals
- Problem-solving
- User input handling
- Program logic building
- Writing clean and structured code
